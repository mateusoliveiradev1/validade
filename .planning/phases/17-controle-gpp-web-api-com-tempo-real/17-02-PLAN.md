## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Central DB to GPP queue | The database is the source of truth for queue/status/saldo; web snapshots are read models only. |
| Idempotency receipt to mutation effect | A retry must return the original outcome without repeating baixa, movement, or audit side effects. |
| Post-baixa records to corrections | Finalized facts must be corrected through linked administrative paths, not overwritten. |
