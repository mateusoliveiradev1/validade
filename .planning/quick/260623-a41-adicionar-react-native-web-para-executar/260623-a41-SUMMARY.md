# Quick Task 260623-a41: Habilitar Expo Web local

O Expo Web do Validade Zero inicia em `http://localhost:8081` e renderiza a tela de acesso sem erros no navegador.

- Adicionada `react-native-web@^0.21.2`, a versao compativel indicada pelo Expo SDK 56.
