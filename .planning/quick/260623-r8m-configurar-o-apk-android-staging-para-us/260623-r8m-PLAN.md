# Quick Task 260623-r8m: Configurar APK Android staging com API publicada

## Goal

Preparar o build Android staging para usar a API Cloudflare Workers publicada em vez do placeholder/local, mantendo o projeto pronto para gerar APK interno via EAS.

## Tasks

1. Validar que a API staging publicada responde publicamente em `workers.dev`.
2. Garantir que `staging` é um ambiente válido no schema compartilhado de configuração.
3. Configurar `apps/mobile/eas.json` para embutir `EXPO_PUBLIC_API_URL` no perfil `staging`.
4. Rodar verificações proporcionais: typecheck/testes do pacote afetado e config EAS.
5. Gerar build Android staging interno quando a configuração EAS estiver pronta.

## Verification

- API pública responde via HTTPS.
- `@validade-zero/config` aceita `VALIDADE_ZERO_APP_ENV=staging`.
- `eas config --profile staging --platform android` mostra a URL da API no ambiente do build.
