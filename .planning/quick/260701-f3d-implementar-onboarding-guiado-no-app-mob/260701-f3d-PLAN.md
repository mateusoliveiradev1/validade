# Implementar onboarding guiado no app mobile

## Objetivo

Melhorar a orientacao dentro do app mobile com um onboarding guiado operacional, separado do fluxo de convite/login. O guia deve levar o operador ao primeiro valor real: preparar a loja, registrar um lote fisico, conferir Hoje e entender quando o turno pode ser fechado com seguranca.

## Escopo

- Adicionar uma tela mobile de "Primeiros passos" com etapas acionaveis.
- Adicionar uma entrada contextual na tela Hoje para abrir o guia.
- Manter `Registrar lote` como acao principal quando nao houver tarefa ativa.
- Preservar a verdade operacional: loja vazia nao significa area segura sem leitura central e conferencia fisica.
- Atualizar a identidade da build mobile se a entrega exigir novo APK instalavel.

## Verificacao

- Testes focados do onboarding e da tela Hoje.
- `pnpm.cmd check`.
- Build Android staging/APK quando a mudanca mobile estiver validada.
- Deploy staging de API/web apenas se a identidade de build publicada precisar acompanhar o novo APK.
